## Node.JS Programming Task

### Technologies used:

Server side:
1. Node.js
2. Express.js
3. Mongoose for MongoDB
4. Socket.io

Client side:

1. Vue.js
2. Socket.io

### How to run:

1. Go to server directory and seed: `npm run seed`

2. Then start server: `npm start`

3. Go to client directory and run: `npm start`
